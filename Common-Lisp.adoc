== Tips and tricks

=== Getting current working directory

* Avoid +*default-pathname-defaults*+, it's implementation-dependent.
* (SBCL) SBCL: +(sb-posix:getcwd)+.

[source,lisp,linenums]
----
;; With cl-cwd
(cl-cwd:get-cwd)

;; With osicat
(osicat:current-directory)
----
